## 🚀 Project Overview
This project is a collection of Python scripts implemented in a Jupyter/Colab environment. It covers three progressive levels of programming logic:

### 1. Factorial Computation (Level 1)
Calculates the factorial of a given positive integer. 
* **Logic:** Uses an iterative loop to compute the product of all integers from 1 up to $n$.
* **Edge Case Handling:** Includes checks for negative integers where factorials are undefined.

### 2. Fibonacci Sequence Generator (Level 2)
Generates a sequence of numbers where each number is the sum of the two preceding ones, starting from 0 and 1.
* **Input:** User-defined number of terms.
* **Output:** A list representing the sequence (e.g., for 7 terms: `[0, 1, 1, 2, 3, 5, 8]`).
<img width="869" height="526" alt="image" src="https://github.com/user-attachments/assets/98a69f26-f392-4d95-ab76-38fd4c8ae41b" />

### 3. CLI Calculator (Level 3)
A modular Command Line Interface (CLI) calculator that performs basic arithmetic operations.
* **Features:**
    * Addition, Subtraction, Multiplication, and Division.
    * Modular design using dedicated functions for each operation.
    * Error handling for **Division by Zero**.
